How to Enter Cheats in the Browser Version

Playing Vice City in the browser works exactly the same as the PC version for cheat codes. Just type the cheat code on your keyboard at any time during gameplay — there's no menu to open. You'll see a brief confirmation message on screen if the code was accepted.

A few things worth knowing:

  • You don't need to pause the game first — just type while playing.
  • Codes are not case-sensitive. Type them in capitals or lowercase, it works either way.
  • If a cheat doesn't register, try clicking the game canvas first to make sure it has keyboard focus, then type the code again.
  • Vehicle spawn cheats will place the vehicle directly in front of Tommy, so make sure there's space.
  • The PANZER (tank) cheat is powerful but can get stuck in tight alleys — spawn it somewhere open.
